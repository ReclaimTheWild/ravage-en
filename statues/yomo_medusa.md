# Yomo Medusa

The Yomo Medusa is a curious stone statue, designed to fling fireballs in four directions. When it detects a foe using an attack directly in line with one of those holes, it uses Yomo Fire to fire all four directions at once. If Heroes are careful not to attack while in line with it, then it will never actually cause them any harm.
